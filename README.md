# BookCourier -- Library-to-Home Delivery System

BookCourier is a modern library logistics platform that allows users to
request book pickup and delivery from nearby libraries. It helps
students, researchers, and general readers borrow and return books
without physically visiting the library.

## Live Links

- Client (Frontend): https://book-courier-sz.vercel.app/
- Server (Backend): https://book-courier-server-sz.vercel.app/

## Purpose

The purpose of this project is to simplify the borrowing and returning
process of library books through an online delivery management system,
improving accessibility and convenience for all library users.

## Key Features

- User account creation and login
- Browse books from multiple libraries
- Request book pickup or home delivery
- Track booking status (Pending → shipped → Delivered)
- Track booking payment status (Paid → Unpaid)
- View delivery and order history
- Admin/librarian dashboard to manage books and requests
- Map view using Leaflet & React Leaflet
- Responsive UI and smooth UX
